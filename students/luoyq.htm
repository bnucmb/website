RESEARCH

Genome-wide analysis of overlapping genes in bacteria genomes (~170 species) tries to ascribe an evolutionary relationship among these species by constructing a satisfactory phylogenetic tree.

Implementing a system for data extraction, and applying data mining techniques on the datasets collected, we constructed a database (BPhyOG) which collects all kinds of features of overlapping genes for Bacterial genomes and a web platform for inferring phylogenomic tree for any set of genomes of interest.

For a set of bacteria genomes, according to the phylogenetic tree inferred using our method, we try to study some special evolutionary events during the process of the organisms' life.
